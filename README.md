# Google Cybersecurity Certificate Portfolio

This repository contains beginner-level cybersecurity portfolio projects completed as part of the Google Cybersecurity Professional Certificate. These projects demonstrate my foundational skills in security auditing, risk assessment, Linux permissions, SQL filtering, incident documentation, network traffic analysis, vulnerability assessment, and basic Python automation.

## Projects Included

### 1. Internal Security Audit - Botium Toys

**File:** `internal-security-audit-botium-toys.pdf`

Completed an internal security audit for Botium Toys, a fictional retail business. This project includes a controls assessment, compliance checklist, risk awareness, and security recommendations to improve the organisation’s security posture.

**Skills demonstrated:** Controls assessment, compliance review, risk awareness, security recommendations.

### 2. Linux File Permissions

**File:** `linux-file-permissions.pdf`

Used Linux commands to check and update file and directory permissions. This project demonstrates how to inspect permissions, understand permission strings, identify hidden files, and modify access using `chmod`.

**Skills demonstrated:** Linux commands, file permissions, access control, hidden files, `chmod`, `ls -la`.

### 3. SQL Security Filters

**File:** `sql-security-filters.pdf`

Used SQL queries to retrieve specific security-related information from a database. This project includes filtering after-hours failed login attempts, login attempts on specific dates, activity outside Mexico, and employee device information.

**Skills demonstrated:** SQL filtering, `WHERE`, `AND`, `OR`, `NOT`, `LIKE`, wildcard filtering, security investigation.

### 4. Incident Handler's Journal

**File:** `incident-handlers-journal.pdf`

Documented multiple cybersecurity investigation activities, including a ransomware incident, packet capture analysis, tcpdump practice, Wireshark usage, and suspicious file hash investigation with VirusTotal.

**Skills demonstrated:** Incident documentation, 5 W’s analysis, ransomware investigation, packet analysis, tcpdump, Wireshark, VirusTotal.

### 5. Network Traffic Analysis: DNS and ICMP

**File:** `network-traffic-analysis-dns-icmp.pdf`

Analyzed DNS and ICMP traffic to identify a network issue involving UDP port 53. The report explains the problem, likely cause, investigation actions, and recommended next steps.

**Skills demonstrated:** DNS analysis, UDP, ICMP errors, port 53, tcpdump analysis, network troubleshooting.

### 6. Incident Report Using NIST CSF

**File:** `incident-report-nist-csf.pdf`

Created an incident report using the NIST Cybersecurity Framework. The report covers an ICMP-based denial-of-service attack and organizes the response using Identify, Protect, Detect, Respond, and Recover.

**Skills demonstrated:** NIST CSF, incident response, DoS analysis, firewall control review, recovery planning.

### 7. Security Incident Report: Brute Force Attack

**File:** `security-incident-report-brute-force.pdf`

Documented a website compromise caused by a brute force attack. The report identifies involved network protocols, explains the incident, and recommends remediation such as multi-factor authentication and login attempt limits.

**Skills demonstrated:** Incident reporting, brute force analysis, DNS, TCP, HTTP, IP, MFA recommendation, security remediation.

### 8. Vulnerability Assessment Report

**File:** `vulnerability-assessment-report.pdf`

Completed a vulnerability assessment for a database server. This project includes a system description, scope, purpose, risk assessment, threat sources, likelihood, severity, and remediation strategy.

**Skills demonstrated:** Vulnerability assessment, risk analysis, threat identification, NIST SP 800-30, access control, remediation planning.

### 9. Python Allow List Automation

**File:** `update-allow-list.py`

Created a Python script to update an allow list by removing specific IP addresses from a file. This project demonstrates basic Python automation for a security-related access control task.

**Skills demonstrated:** Python, file handling, lists, loops, conditional logic, simple security automation.
